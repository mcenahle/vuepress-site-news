# 个人开发测试网站关于不再支持微信内置浏览器打开的通知

[个人主页网站](https://www.mcenahle.org.cn) 已完成工信部 ICP 备案及公安机关备案，网站本身不存在任何安全风险。但由于微信内置浏览器当前对本网站提示“无法确认该网页的安全性，请谨慎访问”，可能对访问体验造成困扰与误解。

为避免给访客带来不必要的困扰，并确保良好的访问体验，[个人主页网站](https://www.mcenahle.org.cn) 和子域名（如 [https://blog.mcenahle.org.cn](https://blog.mcenahle.org.cn)）自即日起不再支持在微信内置浏览器中访问。

如需正常访问，请使用以下方式打开：

- 在系统浏览器中打开（如 Safari、Chrome、Edge 等）；或

- 将链接复制到浏览器中访问。

由此给您带来的不便，敬请谅解。感谢您的理解与支持！
